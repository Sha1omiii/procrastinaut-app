ProcrastinAut

Collaborate with your team and Plan your to do tasks

How to Use

  1. Create a Project: Start by creating a new project for your team. This will be the workspace where all tasks and ideas are organized.
  2. Add To-Do Tasks: Once your project is set up, you can add tasks. Fill in the task details, assign deadlines, and set reminders if needed.
  3. Track Task Completion: Mark tasks as completed by checking the box next to them. Completed tasks will appear with a strikethrough.
  4. Suggest Improvements: Team members can suggest improvements or additions to tasks by submitting suggestions under each task.
  5. Vote on Tasks: Use the voting feature to prioritize tasks. The tasks with more votes get higher attention from the team.
  6. Edit or Delete Tasks: If you need to make changes or remove a task, use the edit and delete buttons to manage tasks efficiently.
  7. Invite Team Members: Invite team members to your project and collaborate on tasks together. They will be able to add, vote, and suggest tasks.
  8. Chat with Your Team: Use the real-time chat feature to discuss tasks and collaborate more effectively.
